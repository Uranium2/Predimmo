1- télecharger la derniere version de python: https://www.python.org/downloads/ ou verifier la
derniere version:cmd -> py

2- installer pip si besoin: 
	- https://bootstrap.pypa.io/get-pip.py enregistrer ce fichier
	- dans cmd, aller à l'endroit du fichier
	- éxécuter py get-pip.py
3- installer django: dans cmd à l'endroit du script pip -> pip install Django==3.0.4
4- dans le prompy python:
>>import django
>>print(django.get_version())

5- installer l'environnement virtuel: https://stackoverflow.com/questions/55142774/how-to-activate-virtual-environment-in-django

CREER UN PROJET

1- Dans cmd, aller dans le repo souhaité: django-admin startproject mysite
2- Avec un éditeur de texte, ouvrir le projet :)
